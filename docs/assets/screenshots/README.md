Regenerate the complete storyboard when recording submission media:

```bash

```

The capture launcher is intentionally no-key. Its build and runtime blank provider credentials, model overrides, and the speech secret, so its character and Case Brief frames must be described as authored fallback behavior, not live GPT-5.6 output.
